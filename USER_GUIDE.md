# SpreadsheetAnalyzer 快速使用指南

## 🚀 快速开始

### 第一次运行

1. **启动程序**
   - 双击 `SpreadsheetAnalyzer.exe`
   - 或在Qt Creator中点击运行按钮

2. **打开数据文件**
   - 方式1: 文件 → 打开 → 选择 `test_data.csv`
   - 方式2: 直接拖拽 `test_data.csv` 到窗口中
 
   - 数据自动显示在表格中
   - 第一行是表头（月份、销售额、成本、利润、客户数）
   - 12行数据（整年数据）

---

## 📊 数据排序 ⭐

### 如何排序
1. **点击列标题** (例如"销售额")
   - 第一次点击：升序排列 (小→大)
   - 第二次点击：降序排列 (大→小)
   - 显示排序指示器 (▲/▼)

2. **支持的排序类型**
   - 数值列：按数值大小排序
   - 文本列：按字母顺序排序

### 实际应用
```
示例：按"销售额"排序
- 点击"销售额"列标题
- 数据从最小值(120000)到最大值(230000)排序
- 再次点击，顺序反转
```

## 🔢 统计分析

### 打开统计对话框
```
工具 → 统计分析
```

### 描述性统计标签页

#### 选择要分析的列
1. 点击"选择列"下拉框
2. 选择"销售额"
3. 点击"计算统计量"按钮

#### 查看结果
```
========== 描述性统计结果 ==========

【集中趋势】
  样本量: 12
  均值: 172875.0000
  中位数: 170500.0000
  众数: 120000.0000

【离散程度】
  最小值: 120000.0000
  最大值: 230000.0000
  极差: 110000.0000
  方差: 103557142.8571
  标准差: 10176.3331

【分布形状】
  偏度: 0.32 (略右偏)
  峰度: -1.23 (平峰分布)
```

#### 统计量解读
- **均值**: 平均销售额
- **中位数**: 销售额中间值
- **偏度**: >0 右偏，<0 左偏，≈0 对称
- **峰度**: >3 尖峰，<3 平峰，=3 正态分布

## 📈 数据预测

### 切换到预测分析
```
点击"预测分析"标签页
```

### 预测步骤

#### 1. 移动平均预测
```
参数设置：
- 选择列: "销售额"
- 预测方法: "简单移动平均"
- 窗口大小: 3 (使用前3个数据)
- 预测期数: 3 (预测未来3个月)

点击"执行预测"
```

#### 2. 指数平滑预测
```
参数设置：
- 平滑系数(α): 0.3 (0.1-0.9)
- 预测期数: 3

点击"执行预测"
```

#### 3. 线性回归预测
```
参数设置：
- 预测期数: 3

点击"执行预测"
```

### 查看预测结果

## 📉 数据可视化

### 切换到图表标签
```
点击"图表"标签页
```

### 创建图表

#### 1. 选择图表类型
```
图表类型下拉框 → 选择"柱状图"
```

#### 2. 查看图表
- 图表自动显示
- 可以修改标题
- 可以导出图片

#### 3. 支持的图表类型

| 图表类型 | 适用场景 | 数据要求 |
|----------|----------|----------|
| 柱状图 | 数据对比 | 多列/单列 |
| 折线图 | 趋势展示 | 单列 |
| 散点图 | 相关性 | 两列 |
| 饼图 | 占比分析 | 单列 |
| 箱型图 | 分布特征 | 单列 |

### 导出图表
```
点击"保存图片"按钮
选择保存位置和格式 (PNG/JPG)
点击"保存"
```

## ⚙️ 设置

### 打开设置对话框
```
工具 → 设置
```

### 常规设置
- **自动保存**: 定时保存文件
- **文件重开**: 启动时打开上次文件
- **退出确认**: 关闭前确认

### 视图设置
- **字体大小**: 8-24pt
- **颜色主题**:
  - 默认 (白色背景)
  - 深色 (护眼模式)
  - 柔和 (淡雅风格)
  - 鲜艳 (高对比度)
  - 商务 (专业风格)

### 图表设置
- **启用动画**: 图表动画效果
- **动画时长**: 100-2000ms
- **默认图表类型**: 选择的默认类型

## 📝 常见问题

### Q1: 如何加载Excel文件?
A: 当前版本Excel支持需要额外配置。建议使用CSV格式，或安装Qt XLSX库。

### Q2: 如何保存分析结果?
A: 可以将图表导出为图片，或将数据导出为CSV文件。

### Q3: 数据太大怎么办?
A: 当前版本适合中小规模数据（<10000行）。大数据量建议使用专业工具。

### Q4: 如何处理日期数据?
A: 确保CSV中日期格式统一（如YYYY-MM-DD），程序会识别为文本。
